# 🍜 천안/아산 모범 음식점 정보 시스템 (C-TCP/IP)

> **선문대학교 컴퓨터공학과 C응용프로젝트**
> C언어 콘솔 기반의 TCP 통신을 이용한 레스토랑 정보 조회 및 별점 관리 시스템입니다.

---

## 😋 프로젝트 개요
본 시스템은 충청남도 공공 데이터를 활용하여 사용자가 천안/아산 지역의 유명 레스토랑을 검색하고, 직접 별점을 내리거나 정보를 관리할 수 있는 클라이언트 서버 구조의 응용 프로그램입니다.

## 🛠 기술 그리드
* **언어:** C
* **네트워크:** WinSock 2.2 (TCP/IP 소켓 통신)
* **데이터 구조:** 구조체 기반 연결리스트 (Linked List)
* **환경:** Visual Studio 2022
* **기능:** 멀티스레딩 서버 (다중 접속 지원)

## ✨ 핵심 기능
1. **사용자 인증**: `users.txt` 기반 로그인 시스템 (최대 3회 시도 제한)
2. **레스토랑 조건 검색**: 지역(천안/아산), 메뉴별 키워드 필터링
3. **정보 관리(CRUD)**: 레스토랑 정보의 추가, 수정, 삭제 기능
4. **사용자 참여**: 별점 평가 및 평균 별점 산출 기능
5. **데이터로 백업**: 프로그램 종료 시 모든 데이터를 `txt` 파일로 백업

## 📂 파일구조
* `server.c`: 클라이언트의 요청을 처리하고 데이터를 관리하는 멀티스레드 서버
* `client.c`: 사용자 인터페이스(UI)를 제공하는 클라이언트 프로그램
* `restaurant.txt`: 공공 데이터베이스 기반 레스토랑 정보 저장 파일
* `users.txt` / `rating.txt`: 사용자 계정 및 별점 데이터 파일
* `docs/`: 개발 계획서 및 요구사항 분석서 PDF 문서
